### Intro
In this exercise you have an existing terraform repo, that needs expansion.

#### Task 1
Look over the repository and describe what it does

#### Task 2
Expand the code to change the instance type to "m5.xlarge"

#### Task 3
Expand the code to create another instance, starting from same AMI, same size, but different name

#### Task 4
Expand the code to create a new S3 bucket.
- bucket name: "company-log-storage"
- a new tag called "Department" with value "Engineering"
- for this bucket, enable all public access

#### Task 5
What can be improved so that multiple users can contribute simultaneously without the fear of destroying other people's work.
